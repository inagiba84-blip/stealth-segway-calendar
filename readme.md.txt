# Календарь мероприятий Stealth & Segway

Одностраничное приложение для добавления мероприятий в Google Календарь и Google Таблицы.

## Настройка
1. Создайте Google Таблицу для хранения данных
2. Настройте Google Apps Script (код в инструкции)
3. Замените SCRIPT_URL в index.html на ваш URL скрипта

## Использование
Откройте index.html в браузере или разместите на GitHub Pages.
